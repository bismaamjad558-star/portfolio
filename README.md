# Personal Portfolio Website

## Project Overview
This is a responsive personal portfolio website built using React.js and Tailwind CSS. It showcases my skills, projects, contact information, and professional profile.

## Features
- Responsive Design
- About Section
- Skills Section
- Projects Section
- GitHub Links
- Contact Section
- Download CV
- Mobile Friendly

## Technologies Used
- React.js
- JavaScript
- HTML5
- CSS3
- Tailwind CSS
- Vite
- Git & GitHub

## Installation Steps

1. Clone the repository:

```bash
git clone https://github.com/bismaamjad558-star/portfolio.git
```

2. Open the project folder:

```bash
cd portfolio
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

5. Open the application in your browser:

```text
http://localhost:5173
```
